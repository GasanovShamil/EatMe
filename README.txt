# EatMeIfYouCan

Logiciels nécessaires :
  MySQL v5.1 ou supérieure.
  JRE v1.8 ou supérieure.
  Eclipse Mars v1 ou supérieure.

Prérequis :
  Importer le projet dans Eclipse.
  Lancer le script CreateDB.sql dans la console MySQL.

Serveur :
  Lancer la classe Server.
  Le port par défaut est "9443", pour le changer modifiez sa valeur dans le fichier src/config.xml.
  Si un client est lancé depuis la même machine que le serveur, l'adresse "localhost" fonctionne.

Client :
  Pour la version console, lancer la classe ClientConsole.
  Pour la version graphique, lancer la classe ClientGraphic.
